## 🌅 朝（07:00-08:30 / 約 15 分）

```

```

```
[ ] AI シナリオを参考に裁量判断で売買

```

## 🌆 引け後（15:50-16:30 / 約 15 分）※最小必須のみ

```
[ ] 1. templates/result_template.md をコピー → results/YYYY-MM-DD_result.md
[ ] 2. 以下の数値だけは必ず記入:
      ・meta: 始値 / 高値 / 安値 / 終値 / 変化 pt
      ・scenarios_predicted: A/B/C どれが当たったか (fired: true/false)
      ・trades_executed: 当日売買全件
      ・pnl_total
[ ] 3. 保有継続ポジションを末尾にメモ
```

※ review.md は余裕ある日だけ（初週はスキップ OK）
